# ValueUp RESTful 강의 서버 샘플
단말과의 통신을 처리하는 서버 샘플 입니다.

## 요구사항  
- JDK 1.6+
- Maven 3+
- MariaDB(MySQL)

## 환경설정
### JDK 설치
[Oracle JDK 다운르도 페이지](http://www.oracle.com/technetwork/java/javase/downloads/index.html)에서 JDK 다운로드 및 설치.  
1.6 버전의 경우는 지원 중단되었으므로 OS 환경에 따라 1.7 이상의 버전을 설치.
